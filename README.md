# Yuutube
JavaFX Desktop Application Youtube Clone.

User can upload video to their profile.
User can delete their own video.
User can view stats of their video(likes, views...)
User can search for videos and watch them.

Frontend: JavaFX
Backend: Java
DB : MySQL
